\begin{abstract}
We describe a software workflow for estimating time-varying inflow
parameters in urban-flow large-eddy simulations from sparse
velocity observations.  The workflow runs an ensemble smoother with
multiple data assimilation (\ESMDA{}) over a sequence of short
windows that together cover a long simulation horizon.  Each
window's posterior parameter ensemble is propagated into the
prior of the next window through a generative model of the parameter
trajectory.  Four such models are implemented behind a common
interface, with a critically-damped AR(2) relaxation as the default.
The data-assimilation logic is solver-agnostic and supports three
urban-flow forward models: a custom lattice-Boltzmann implementation,
uDALES, and PALM.  This report documents the methodology and serves
as the basis for an upcoming academic article.
\end{abstract}
